# 🏥 Shifa Home Health Care

A premium, medical-grade home health monitoring and clinical management platform built with Flutter.

---

## 🚀 Technology Stack
* **Framework**: Flutter (Material 3 Design)
* **State Management**: Flutter Riverpod
* **Routing**: GoRouter (Shell Navigation)
* **Database & Auth**: Firebase Firestore & Firebase Authentication

---

## 📦 Core Modules
* **🔒 Secure Authentication**: splash screen flow and staff login portal.
* **📋 Patient Management**: manual MRN patient registration, search, and profiles.
* **🩺 Real-Time Vitals**: dynamic grid displaying latest Spo2, pulse, temp, BP, and blood sugar.
* **💧 I/O Fluid Logs**: automated intake/output fluid balance calculations with Firestore sync.
* **💊 Medication Tracking**: active prescriptions, doses, frequencies, and scheduler.
* **📝 Notes & Clinical History**: nursing shift logs, timeline updates, and records.
* **📄 Reports**: PDF summaries and shift logs exporter.

---

## 🛠️ Quick Start

### 1. Install Dependencies
```bash
flutter pub get
```

### 2. Run the Application

flutter run
